Restore Instructions

Before restoring:

  1. A safety snapshot of the current database is automatically created before any restore operation.
  2. Validate the target backup before restoring to confirm its integrity.
  3. Restoring replaces the live database - all changes made after the backup date will be lost.
  4. Only validated (PASS) backups can be used for restoration.
